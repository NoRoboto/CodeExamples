## Pre-conditions
#### <i class="icon-list"></i> We need

> - [Nodejs](https://nodejs.org/en/) >= 6.1
> - [NPM](https://www.npmjs.com/)
> - (Optional) [Yarn](https://yarnpkg.com/en/)

## Installation
```
yarn or npm install
```
## Run
```
yarn start or npm start
```
